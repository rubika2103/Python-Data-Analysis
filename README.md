# Python Data Analysis & EDA Project

This project demonstrates data cleaning, exploration, visualization, and insight extraction using Python. The analysis was performed using Pandas, NumPy, Matplotlib, and Seaborn inside a Jupyter Notebook.

- Handled missing values

## Exploratory Data Analysis (EDA)
The analysis explores how different attributes impact **cup score** and overall coffee quality.

### Key EDA steps:
- Summary statistics for all features  
- Distribution of coffee quality scores  
- Country-wise coffee quality comparison  
- Processing method analysis  
- Correlation heatmap  
- Flavor vs Aroma vs Sweetness comparison  
- Scatter plots for quality attributes

## Key Insights
- Aroma, flavor, and aftertaste show strong correlation with final cup score.  
- Arabica beans generally score higher than Robusta.  
- Natural and washed processing methods tend to deliver better quality scores.  
- Coffee from certain regions consistently achieved higher flavor and acidity ratings.  
- Defects and moisture level significantly affect coffee grade.
